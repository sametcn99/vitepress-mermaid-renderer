## Reporting a Vulnerability

We take the security of `vitepress-mermaid-renderer` seriously. If you have found a security vulnerability in this project, please help us fix it by reporting it responsibly.

### How to Report

1. **Do not open a public issue.** Publicly reporting a vulnerability can put the community at risk.
2. Please verify the vulnerability by reproducing it with the latest version of the package.
3. Send a detailed report to **sametcn99@gmail.com**. Please include:
   - A description of the vulnerability.
   - Steps to reproduce the issue.
   - Any relevant proof-of-concept code or screenshots.

### Response Timeline

- We will acknowledge your report within **48 hours**.
- We will provide an estimated timeline for the fix within **1 week**.
- Once the fix is verified and released, we will publicly acknowledge your contribution (unless you prefer to remain anonymous).

Thank you for helping keep the community safe!
